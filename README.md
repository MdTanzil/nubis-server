##  Project features

* This project use Express js and mongoDb for impliment api




## API Reference

#### Create  a product

```http
  POST /products
```



#### Get Product Brand name wise

```http
  GET /products-brand/${brandName}
```

#### Get a Single Product

```http
  GET /products/${id}
```
#### Update a Single Product

```http
  PUT /products/${id}
```


#### Add Product to Cart

```http
  POST /Cart/
```
#### Get All Cart
```http
  GET /cart
```
#### DELETE a cart Item
```http
  DELETE /cart/${id}
```
#### Get Brand Images
```http
  GET /brand-images/${name}
```
